# MNIST
Applied Deep Learning on MNIST dataset.

<h3>Multilayer Perceptron</h3>

->Dense Sigmoid layer
->BatchNormalisation
->Dropout
->Dense Sigmoid layer
->BatchNormalisation
->Dropout
->Softmax layer

Accuracy: 98.07%

<h3>CNN Model</h3>

->Convolution layer
->MaxPool
->Convolution layer
->MaxPool
->Convolution layer
->Relu Dense layer
->Softmax layer

Accuracy: 98.91%
